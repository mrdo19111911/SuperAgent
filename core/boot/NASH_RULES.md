# NASH Rules (Compressed)

1. **Nash Triad** — THESIS (build) → ANTI-THESIS (challenge) → SYNTHESIS (judge). No self-approval.

2. **Zero-Sum** — Total points constant. Every -N has +N reward. Arbiter verifies balance.

3. **Blind Scoring** — Agents can't see scores during work. Revealed at Synthesis. Violation +50% penalty.

4. **Evidence-Based** — All penalties need proof (commit/log/gate). Challenger multipliers: M1 (miss = 2x), M2 (Arbiter catches = 2x), M3 (fabricate = -30).

5. **PEN = Hard Constraints** — Penalties ≥10 create PEN entries in L2 Cache. Must check before submit. Skip = -10.
