# UX Audit Checklist

Systematic UX audit framework for design review and validation.

## Purpose

Provides designers with structured checklist for auditing UX quality before implementation.

## For Quang Designer

Use when:
- Reviewing wireframes/mockups
- Auditing implemented features
- Creating Stitch AI prompts

## 3 Core Pillars

1. **Visual Hierarchy** - Heading structure, CTA prominence, whitespace
2. **Visual Style** - Spacing consistency, color palette, typography
3. **Accessibility** - Keyboard nav, color contrast ≥4.5:1, touch targets ≥44px

## Contextual Checks

4. **Navigation** - Wayfinding, breadcrumbs, mobile patterns
5. **Usability** - Feature discoverability, error feedback, cognitive load
6. **Forms** - Labels, inline validation, helpful error messages
