| Principle | Application |
|-----------|-------------|
| Emergence | Control Tower cross-platform patterns |
| Feedback | Exception → optimize → reduce |

| Leverage | AI 5-field autofill → 50% faster |

## B2B Context

**Users:** 60% age 45+, presbyopia, mixed tech skill, 4hr train, <5 tickets/30d

**Perf Expectations:**

| Type | Time | UX |
|------|------|----|
| Simple | 100-200ms | None |
| Page | 1-2s | Skeleton @200ms |
| Report | 5-10s | Progress + cancel |
| Bulk | 10-60s | Background + notify |

**Constraints:**
- WCAG 2.1 AA: 4.5:1 text, 3:1 large, colorblind, keyboard, 44px
- Devices: 70% desktop, 20% tablet, 10% mobile + scanner
- Network: Office (broadband), WH (WiFi gaps), Field (3G/4G) → offline-first

## Mental Models by Role

| Role | Focus | View | Key Metrics |
|------|-------|------|-------------|
| Planner | Capacity/Routes/Cost | Board+Map | Cost/km, Load% |
| WH Operator | Zones/Tasks/Sequence | List+Floor | Tasks/hr, Accuracy |
| Coordinator | Timeline/Carrier/ETA | Map+Timeline | On-time%, ETA |
| SC Manager | Demand/Supply/Inventory | Dash+Forecast | Fill rate, Stock-out |
| Marketplace User | Price/Service/Rating | Search+Compare | Price, SLA, Reviews |
| Executive | Exception/Trends/KPI | Control Tower | Cost variance, SLA breach |

## Quantified Examples

- **Hick's:** 160 info (20 carriers×8 pts), 10-15min → 9 info (AI 3×3), 30s = 95%↓
- **Miller's:** 15 cols overload → 7 cols = 50% faster scan
- **Fitts's:** Top-right btn 800px → card btn 0px, 100×/d = 5-8min/d saved

**Sources:** Norman (mental models), Krug (usability), Meadows (systems), WCAG 2.1
